# 1.1 术语
平台是由一个或多个元件组成的单一集成电路。
一些组件可能是RISC-V核心，而其他组件可能有不同的功能。
通常它们都将连接到一个单一的系统总线。
一个RISC-V核心包含一个或多个硬件线程，称为harts。
hart的DXLEN是其最宽支持的XLEN，忽略misa中MXL的当前值。
